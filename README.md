# Meeting Action Tracker

会議内容からアクションアイテムを整理・管理するための  
シンプルな Web アプリケーションです。

FastAPI + Jinja2 を用いて、  
「入力しやすさ」「一覧性」「実務で使えること」を重視して設計しました。

---

## 概要

- 会議を登録
- 会議ごとにアクション（TODO）を管理
- 完了／未完了の切り替えが可能
- 小規模チームや個人利用を想定したシンプル設計

---

## 主な機能

- 会議一覧表示
- 会議の新規作成・削除
- 会議ごとのアクション管理
- アクションの完了／未完了切り替え
- シンプルなUIで直感的に操作可能

---

## 使用技術

- Python
- FastAPI
- Jinja2（テンプレート）
- SQLAlchemy
- SQLite
- HTML / CSS

---

## 画面構成

- 会議一覧画面
- 会議詳細画面（アクション一覧）

---

## 工夫した点

- FastAPI を API だけでなく、画面描画まで含めて構築
- MVC を意識したディレクトリ構成
- 実務利用を想定し、機能を絞ったシンプル設計
- ローカルDBは `.gitignore` により管理対象外に設定
